# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1Lkh5ACrEM4ZBkO9P4jR2rrZMTyuEz7-7

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

---

## Demo Authentication Codes

For testing and administrative access, use the following codes on the login screen:
- **Student Portal:** `DEMO-STUDENT`
- **Facilitator Portal:** `DEMO-FAC`
- **System Admin Console:** `ADMIN-MASTER`
